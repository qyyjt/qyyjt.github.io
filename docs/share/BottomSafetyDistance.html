  • ios添加底部安全距离 ##

    • ios添加底部安全距离
      • ios去除底部安全距离

      # ios添加底部安全距离

      移动端页面，如iPhoneX iPhone11等手机，页面底部会有个操作条，影响页面操作。我们可以留有一个安全距离。但是移动端设置margin-bottom是默认无效的。此时需要进行额外的配置。

      首先，在index.html的meta标签设置 viewport-fit=cover,user-scalable=no ：

      <meta name="viewport" content="width=device-width,initial-scale=1.0,viewport-fit=cover,user-scalable=no">

      接着给body设置样式（若不生效，可将样式设置在body内一级标签上面）：

      body {
	padding-bottom: constant(safe-area-inset-bottom); /* 兼容 iOS < 11.2 */
	padding-bottom: env(safe-area-inset-bottom); /* 兼容 iOS >= 11.2 */
}

      或者

      body {
  height: 100%;
  height: calc(100vh - constant(safe-area-inset-bottom)); //兼容 IOS<11.2
  height: calc(100vh - env(safe-area-inset-bottom)); //兼容 IOS>11.2
  overflow: auto;
}

      在公司APP项目中，通常添加到body元素上面无法生效，此时可将上述样式追加到根元素上面，如

      <div class="container"></div>

      # ios去除底部安全距离

      1 现象 ：iphoneX以上型号的手机 底部导航栏被黑线遮挡 影响用户体验，如何去掉底部的遮挡。

      2 解决办法：

      第一步设置网页的头部标签 ：viewport-fit=cover ；

       viewport-fit共有三个属性：其中我们适配的关键就是viewport-fit=cover
  contain 可视窗口完全包含网页内容
  cover：网页内容完全覆盖可视窗口
  auto：默认值，跟 contain 表现一致

      <meta name="viewport" content="width=device-width,initial-scale=1.0,viewport-fit=cover">

      第二步：通过媒体查询设置底部的padding-bottom 的值

      .mobile-top-bar {
  padding-top: constant(safe-area-inset-top);
  padding-top: env(safe-area-inset-top);
}
@media only screen and (device-width: 375px) and (device-height: 812px) and (-webkit-device-pixel-ratio: 3), only screen and (device-width: 414px) and (device-height: 896px) and (-webkit-device-pixel-ratio: 3), only screen and (device-width: 414px) and (device-height: 896px) and (-webkit-device-pixel-ratio: 2), only screen and (device-width: 375px) and (device-height: 812px) and (-webkit-device-pixel-ratio: 3), only screen and (device-width: 414px) and (device-height: 896px) and (-webkit-device-pixel-ratio: 3), only screen and (device-width: 375px) and (device-height: 812px) and (-webkit-device-pixel-ratio: 3), only screen and (device-width: 390px) and (device-height: 844px) and (-webkit-device-pixel-ratio: 3), only screen and (device-width: 428px) and (device-height: 926px) and (-webkit-device-pixel-ratio: 3) {
  /*底部适配*/
  .mobile-bottom-bar {
    box-sizing: border-box;
    padding-bottom: constant(safe-area-inset-bottom);
    padding-bottom: env(safe-area-inset-bottom);
  }
}

      第三步：将设置的类名添加到app项目中的父级div

      <div  class="mobile-bottom-bar app"></div>
