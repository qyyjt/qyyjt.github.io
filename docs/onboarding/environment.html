# 公司前端环境介绍

公司现有项目可以大体分为三类： dojo， requirejs + 自写mvc框架，webpack + vue2

# Dojo部分：

Svn： [https://192.168.100.188/svn/HTML5/trunk ](https://192.168.100.188/svn/HTML5/trunk

代表项目： 研究报告 公司新闻

主要工作： 代码维护，包括后台接口更换等

代码打包方式： 手动

打开cmd->cd切换目录到util/buildScripts/文件夹下， 运行：build.bat --profile ./profiles/standard.profile.js

注意： html中注释误删

# Requirejs:

svn: https://192.168.100.188/svn/FaTeam/Demo/trunk/httproot (opens new window)代表项目： 大部分项目

主要插件： slick.grid,ztree 以及 httproot\jquery\amd\widgets目录下自定义插件

部分项目使用 vue（version 1） vue-router(慧搜)

代码打包方式： 上传svn后自动打包

当发生上传后内网页面有问题时处理：

1 检查 http://192.168.100.59/ 页面版本中是否有更新记录

2 检查代码是否有遗漏未上传

3 可以手动打包查看是否打包过程出现问题
