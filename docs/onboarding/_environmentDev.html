# 一、公司产品介绍

公司产品主要包括大智慧PC终端（127,321），企业预警通APP项目（北京银行分支项目）。

PC终端现有项目可以大体分为三类： dojo， requirejs + 自写mvc框架，webpack + vue2

其中，dojo与vue2项目需要手动打包。Mvc框架项目中，有使用vue0.12版本和vue1.0版本。上传svn后自动打包，但需验证成功。自动打包上传后，会自动发布到终端svn上，一定时间后自动升级。

企业预警通有多个分支（内外网svn，线上源码svn，北京银行svn）详见开发文档。

3). Webpack+vue2项目

手动打包
