# App项目说明

# 一、脚手架说明

Svn： https://192.168.100.188/svn/FaTeam/Demo/tags/mobile/vue-scaffold (opens new window)

Build： webpack 配置

Dll: 第三方js静态缓存(webpack用)

Src： 源码

Assets： 图片、字体及音频文件

Components: 公共组件（详见： https://192.168.100.188/svn/FC_APP/开发相关/前端页面/app页面组件、插件与指令说明.xlsx）

注意： ui打头的公共组件，如果单个项目中需要修改的话。请复制一份修改文件名。

Page： 项目页面 styles: 公共css(normalize引用，取消按钮链接上的文字选中等)及方法（最细边框线等）

Utils

http: 对ajax进行的封装(提供msgbox及alert)

appAssist.js: 与原生的交互方法

Dom.js：页面尺寸初始化及水印等公共方法

Url.js url处理

...

Gulpfile.js: gulp命令配置

Build: 项目打包 checkout: 更新内网服务器svn代码

Checkin: 提交到内网服务器svn

Upload： 上传文件到内网http服务器

... (其它babel、eslint、prettier配置文件等)

Ps： 内网服务器整点更新，http服务器上的内容会被覆盖。

# 二、jenkins服务器：

自动打包发布服务器： 地址： http://10.15.97.192:8090 (opens new window)

新项目可以按jenkins服务器上的原有任务进行配置进行修改
